Mempelajari Domain dan Hosting

Mempelajari Domain dan Hosting dengan lebih dalam

Hosting dan Domain adalah objek yang berbeda, walaupun bisa menjadi satu kesatuan. Dalam artikel ini akan di jelaskan mengenai perbedaan hosting dan domain.

Domain

Domain adalah nama yang ketentuannya diatur oleh badan regulasi domain dunia. Nama domain ini yang akan mengarahkan ke alamat IP server hosting, sehingga saat anda mengetikan nama domain, yang anda akses adalah server IP yang dituju. Pada saat anda membeli domain pada dasarnya anda hanya menyewa nama saja, dengan domain saja anda tidak bisa menyimpan data web atau aplikasi web lainnya sehingga saat anda membeli domain tanpa hosting saat domain di akses tidak akan menampilkan apapun. Kesimpulannya, hosting merupakan tempat penyimpanan data, sedangkan nama domain adalah nama atau alamat saja, agar mudah di ingat. Jadi agar web anda bisa di akses, anda harus melakukan pembelian hosting dan nama domain sekaligus. Keduanya bisa di beli terpisah,.

Hosting

Hosting merupakan tempat penyimpanan data atau file komputer. Hosting memiliki bentuk fisik yang biasa disebut dengan server. Server itu sendiri sama seperti komputer biasa, memiliki Hard disk, RAM, Prosesor, dll. Hanya bentuknya di sesuaikan dengan kebutuhan agar bisa di simpan di rack server. Server hosting ini di simpan di datacenter yang dilengkapi dengan listrik dan backup listrik berupa UPS. Server hosting terkoneksi dengan jaringan internet berkecepatan tinggi agar bisa di akses secara bersamaan secara simultan dari seluruh dunia. Jika hosting di peruntukan untuk website, maka biasanya di sebut dengan Web Hosting. Server untuk web hosting di lengkapi dengan OS dan sofware yang bisa menjalankan aplikasi web, dan mampu beroperasi secara terus menerus. Saat anda membeli / menyewa hosting, yang anda sewa adalah jasa penyimpanan data saja, saat mengakses harus menggunakan alamat IP address.
